# Food Recipe App

🚀 A Full Stack MERN Recipe Sharing Platform for discovering, browsing, and managing recipes through a modern and responsive web interface.

🌐 Live Demo: https://myrecipehub.vercel.app

---

## Overview

Food Recipe App is a full-stack web application built using the MERN stack (MongoDB, Express.js, React, and Node.js). The platform enables users to explore recipes through a clean and responsive interface while demonstrating backend API integration and database connectivity.

---

## Key Highlights

- Full Stack MERN Application
- Responsive User Interface
- MongoDB Integration
- REST API Architecture
- External Recipe API Integration
- Production Deployment on Vercel

## Installation

### Clone Repository

```bash
git clone https://github.com/KANHAIYASINGH337/FoodRecipe.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
